# Nwitter
Twitter (mini)Clone with React and Firebase  
SNS Twitter를 클론 코딩한 프로젝트입니다.  
개인 프로젝트

## 기능 목록
- 회원가입
- 일반 로그인/소셜(google, github) 로그인
- 트윗 CRUD
- 내가 쓴 글 관리
